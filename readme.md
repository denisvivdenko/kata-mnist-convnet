# Project Scope

Model classifies numbers (1-9) from images. The end goal is to deploy trained
model API on a server.

## Tasks:
1. Preprocess data.
2. Train model.
3. Expose API.
4. Deploy on a server.

### Preprocess data.
    1. Load data.
    2. Split data.

### Train model.
    1. Compile model architecture.
    2. Train model.
    3. Evaluate model.

### Expose API.
    1. Expose FastAPI endpoint to be able to submit images.
